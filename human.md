# human.md — Product Intent

## What BreathBetter Is
A mindful breathing application that helps users reduce stress, increase energy, and enhance well-being through guided breathing exercises.

## Target User
- People seeking stress relief and calm
- Meditation/mindfulness practitioners
- Those exploring breathwork for focus, sleep, or anxiety
- Users who value simplicity over feature overload

## Emotional Goals
- **Calm**: The app should feel like a refuge, not another productivity tool
- **Trust**: Clear, honest interface — no dark patterns or manipulation
- **Grounding**: Breathing is intimate; the UX should feel safe and supportive
- **Accomplishment**: Gentle sense of progress without gamification pressure

## Core User Journeys

### 1. First-time user
Home → Practice → Select pattern → Start breathing → Experience calm

### 2. Returning user
Direct to Practice → Start session → Track progress (premium)

### 3. Premium exploration
Practice → See premium prompt → Visit premium page → Activate → Access custom patterns, stats, visualizations

## UX Principles
- Generous whitespace and breathing room
- Subtle animations that aid focus, not distract
- No clutter — only what the user needs in the moment
- Accessibility is foundational, not an afterthought
- Mobile-first, but elegant on desktop
- Dark mode default matches the calm, contemplative vibe

## Non-Goals
- Social features, sharing, leaderboards
- Aggressive upselling or premium gates on core functionality
- AI-generated content or chatbot interactions
- Complex settings or overwhelming customization
- Gamification that creates anxiety about "streaks"
